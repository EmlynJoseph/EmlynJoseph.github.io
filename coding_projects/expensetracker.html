Expense Tracker

Web App

This is a project from the Full-Stack-Engineer path in Codecademy after the Redux Toolkit lesson.
Redux Toolkit is essentially a tool created by the Redux team for developers to simplify and refactor Redux code. It makes it very easy to write Redux applications as it builds the best practices, simplifies most of the Redux tasks, and even prevents many errors. This project uses two of the essential methods of the Redux toolkit, which are createSlice() and configureStore(). Both these methods simplify and refactor the code that was made without using Redux Toolkit.
For this project, most of the code was provided by Codecademy, but my tasks were to refactor two files, budgetsSlice.js and transactionsSlice.js, using the methods from Redux Toolkit.
This app is an expense tracker. You can add funds for any category and make transactions, which will deduct the amount specified from the specified category.
This web app was deployed through Netlify. Click the "Web App" link below the title to view the app.
