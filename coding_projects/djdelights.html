Django Delights

Web App

This project is the final project (Django Capstone Project) for the Build Python Web Apps With Django Skill Path in Codecademy. Django is a powerful web framework and this project strongly enhanced my skills to make interactive and secure django web apps.
I built almost the entire project on my own in my local environment, but I used some help from Codecademy when creatings models for each main component and storing the user inputted data into an organized database. Throughout the project, when I have questions or errors, I use online resources such as Stack Overflow and documentation to answer my question or fix my error. I've made the entire web app responsive in various devices. Also, to make each user's data secure, I've implemented login and signup pages and you can't access the web app without doing that.
Basically, this app resembles a store. You can make new menus of your choice, add new ingredients, add recipe requirements, and even purchase an item. I've also created a page for statistics, which show your revenue, costs, and profit.
