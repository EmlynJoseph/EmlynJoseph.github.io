HTML Cheatsheet

Common HTML Tags

Tag Name Description
<!-- Comment Defines a comment.
<DOCTYPE html> Document Type The first line of a HTML document that defines the document type.
<html> Html Defines an HTML document and contains all the content.
<head> Head Defines the head portion and contains the metadata.
<meta> Meta Defines the metadata.
<title> Title Defines the title which is shown on the page's tab.
<body> Body Contains the content that is displayed in the browser.
<link> Link Defines a link to an external document and the path of the url can be added using the "href" attribute.
<a> Anchor Defines a hyperlink with a url that can be added using the "href" attribute.
<h1> Heading 1 Defines a 1st level heading that contains the most important or main information.
<h2> Heading 2 Defines a 2nd level heading that contains the second most important information.
<h3> Heading 3 Defines a 3rd level heading that contains the third most important information.
<h4> Heading 4 Defines a 4th level heading that contains the fourth most important information.
<h5> Heading 5 Defines a 5th level heading that contains the fifth most important information.
<h6> Heading 6 Defines a 6th level heading that contains the least important information.
<p> Paragraph Defines a paragraph.
<br> Line break Adds a line break.
<ul> Unordered List Defines an unordered list.
<ol> Ordered List Defines an ordered list.
<li> List item Defines a list item inside a <ul> or <ol> that defaults to bullet points for an unordered list and numbers for an ordered list.
<em> Emphasis Emphasizes text in italics, similar to the <i> tag.
<i> Italic Defines text in italics.
<strong> Strong emphasis Defines text with strong emphasis displayed in bold.
<img> Image Defines text with strong emphasis displayed in bold.
<video> Video Defines text with strong emphasis displayed in bold.
<button> Button Creates a clickable button.
<form> Form Creates a form for the user.
<input> Input Creates a field for user input, usually inside a form.
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<th> Table Header Creates a header cell at the top of a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<tfoot> Table Foot The set of rows defining the footer in a table.
