This is the portfolio project at the end of the Create an Advanced Web App with React and Redux Skill Path in Codecademy. This project uses React and Redux...
This project also uses a few API's, Unsplash API, OpenWeatherMap API, and They Said So Quotes API. I interacted with these API's by making asynchronous requests with a URI (Uniform Resource Identifier) with specific queries. Then, I handled the recieved data and extracted the data I needed for my app.Throughout the processes when using APIs, I referred to each of the documentations, Unsplash Documentation, OpenWeatherMap Documentation, and They Said So Quotes Documentation.
Before starting the project, I set a timeline for myself with firm boundaries and deadlines so that I can stay on track. I first created the file structure of the entire app, which gives me a general overview of the app. After that, I worked on each feature of the app, not worrying about styles. As a result, the rendering was scattered across the page. I later configured the Redux store and binded it to the React UI rendering in order to make Redux accessible through the React components. The App componnet was the top level component that rendered each component/feature. Lastly, I worked on the styles.
This was a challenging project and Stack Overflow and the documentations were a great resource throughout the whole project.
