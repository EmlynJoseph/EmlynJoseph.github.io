Inspirational Stopwatch

Web App


This is the portfolio project at the end of the Create an Advanced Web App with React and Redux Skill Path in Codecademy. Instead of making an inspirational to-do app shown on Codecademy, I made an inspirational stopwatch. This project uses React and Redux, two versatile libraries that work really well with each other. React is for the UI and Redux is for managing the state of the app.
This project also uses a few API's, Unsplash API, OpenWeatherMap API, and They Said So Quotes API. I used these APIs by making asynchronous requests with a URI (Uniform Resource Identifier) with specific queries. Then, I handled the received data and extracted the data I needed for my app. Throughout the processes when using APIs, I referred to each of the documentations, Unsplash Documentation, OpenWeatherMap Documentation, and They Said So Quotes Documentation.
Before starting the project, I set a timeline for myself with firm boundaries and deadlines so that I can stay on track. I first created the file structure of the entire app, which gives me a general overview of the app. After that, I worked on each feature of the app, not worrying about styles. As a result, the rendering was scattered across the page. I later configured the Redux store and binded it to the React UI rendering in order to make Redux accessible through the React components. Creating the stopwatch was the easier part and I referred to some resources and tutorials online. The App component was the top level component that rendered each component/feature. I worked on the styles and responsiveness at the end of each feature.
This was a challenging project and Stack Overflow and the documentations were great resources throughout the whole project.
I used Git for version control and regularly updated my changes onto Github. This web app was deployed through Netlify. Click the "Web App" link below the title to view the app.
