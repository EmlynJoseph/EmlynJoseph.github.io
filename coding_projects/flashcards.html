Flashcards

Web App

This is a challenge project from the Full-Stack-Engineer Path in Codecademy at the end of the Redux module.
This project uses React for the UI rendering and Redux for the state management. Both these libraries are interconnected and work really well with each other.
I downloaded the starter code from Codecademy and worked on all the tasks in my local environment. I worked on implementing the app's Redux logic to manage the complex state of this flashcard-quiz app. I also modified all of the styles by changing all the hard-coded and fixed values, adding much more user interactivity, and make the app look a lot more appealing.
This web app resembles a flashcards quiz app. Users can create their own topics, quizzes for those topics, and flashcards for those quizzes. Adding on to this, users can interact with their quizzes by flipping the flashcards over.
This web app was deployed through Netlify. Click the link above to view and interact with the app.
