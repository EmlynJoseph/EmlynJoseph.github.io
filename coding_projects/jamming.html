Jamming (Spotify App)

Web App

This was a large project made using many languages and libraries. This is one of the projects that I had to create and complete in the full-stack-engineer path in Codecademy. The key concepts used to create this web app were React components, passing the state from one component to another, and requests with the Spotify API in order allow users to search the Spotify library, create a custom playlist, and then save it to their Spotify account, all through this website.
When building this web app, the code is organized into several folders. Each one has a React Component, which serves it own purpose, and a CSS file to style it. For example, one component is the search bar, which I linked to the Spotify search. These components are all connected and rendered through the main file. I built most of the project on my own in my local environment, but I struggled a little bit in the last part when creating requests and fetching data through the Spotify API and manipulating it to return songs, artists, or albums in an organized format. For this part, I used videos and tutorials from Codecademy. This web app was deployed through Netlify. You can click the link below to view my project and use it.
