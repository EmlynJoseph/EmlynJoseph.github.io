Passing Thoughts

Web App

This project is from the Full-Stack-Engineer Path in Codecademy after the lesson on React Hooks.
This web app uses two main React hooks: useState and useEffect. React hooks can be used in simple function components that let us manage the internal states of them and handle post-rendering effects. The hooks provide an easier way instead of creating complex classes which are time-consuming when testing and can cause many bugs.
In this project, you can record short thoughts and have a moment to reflect. You can delete the thoughts by pressing the 'x', but each thought will automatically fade away after 15 seconds.
Even though I finished this project in the Codecademy interface, I transferred it to my local environment and deployed it through Netlify. Click the "Web App" link below the title to view the app.
