Random Color Picker

Web App

This project is from the Full-Stack-Engineer Path in Codecademy after the lesson on Components Interacting.
The key concepts used to create this project were components, constructors, states, and props. The purpose of this app is to display new and random color schemes for designers. Clicking the refresh button generates a new color. The new color is rendered to the screen, changing the background each time the button is pressed. If someone finds that color appealing, they can copy the rgb(red, green, blue) values at the top of the screen.
Even though I finished this project in the Codecademy interface, I transferred it to my local environment and deployed it through Netlify. Click the link above to view the web app.
